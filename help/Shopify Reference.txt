The default way to target a specific collection is collections.COLLECTION_HANDLE or collections[COLLECTION_HANDLE]

So if you have a collection called Shirts it will probably have a handle called shirts ( it may be different, you need to check it ) then you call collections['shirts'] and since you want the products you target them like so collections['shirts'].products 


<ul>
	{% for product in collections['shirts'].products limit: 8 %}
	<li>
	<img src="{{ product.featured_image | product_img_url: 'medium' }}" alt="{{ product.title | escape }}" /> 
	<a class="shop-now" href="{{ product.url }}">{{ product.title | escape }} 
	</a> 
	</li>
	{% endfor %}
</ul>
